## Decisões técnicas importantes

- **Proteção de profile admin:** ao salvar cliente, usa `cliente_user_id` (e não `responsavel_user_id`) para não sobrescrever dados do usuário admin logado.
- **Integridade de email:** edge function `wix-client-intake` agora retorna erro se email já existe, evitando silenciosamente sobrescrever um profile existente.
- **UX certificado:** modal abre automaticamente se dados obrigatórios faltarem, evitando erro silencioso.
- **Multi-org:** admin pode trocar de organização diretamente no painel, suportando operação multi-tenant.
