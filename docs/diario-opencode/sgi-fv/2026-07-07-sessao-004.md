- Evitado passar string vazia como UUID em `org_members` query
- Adicionado `actor_user_id` ao insert de `process_events` (not-null constraint)
- Alterado `orgRole` de `'member'` para `'client'` (check constraint)
- Carregados processos reais do banco no dashboard (dbProcesses)
- `create-user` edge function agora retorna erro ao falhar upsert de `org_members`
- Removido protocolo mock (SGI-2026-000) de usuários sem processo
- Validado `processRecordId` antes de anexar documentos (corrige erro 23503)
- SELECT anônimo de organizações liberado para página de registro pública
- Filtradas apenas organizações ativas no `loadOrganizations`
- Valores padrão "Selecione a organização" e "Selecione o tipo..." no modal de criar processo

### 5. Melhorias diversas
- **Version checker**: adicionado sistema de auto-update para PWA mobile
- **Relatório PDF**: botão "Expedir Relatório" na comunicação — gera PDF da conversa com datas, remetentes, mensagens e anexos
- **Double-encoding UTF-8**: corrigido em todas as strings com acentos do AdminDashboard
- **Minifier**: trocado de terser para esbuild para corrigir corrupção UTF-8
- **Permissões**: passado `profile_role` para `resolvePermissions` para correto mapeamento de hierarquia admin
